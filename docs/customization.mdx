# Customization

You can refer to props described below to customize the slider.

## Example

<Playground>
    <Slider
        width={500}
        height={250}
        maxSwipeThreshold={500 * 0.15}
        minSwipeThreshold={40}
        swipeTimeThreshold={200}
        images={[
            "https://www.chewy.com/petcentral/wp-content/uploads/2018/05/lucky-corgi-butts-x-596-444x.jpg",
            "https://s3.amazonaws.com/cdn-origin-etr.akc.org/wp-content/uploads/2017/11/12225919/Pembroke-Welsh-Corgi-On-White-01.jpg",
            "https://upload.wikimedia.org/wikipedia/commons/thumb/f/fb/Welchcorgipembroke.JPG/1200px-Welchcorgipembroke.JPG"
        ]}
        selectorStyle={{
            backgroundColor: "blue",
            height: "50px",
            width: "50px"
        }}
        slideStyle={{
            borderRadius: "100px"
        }}
        selectorContainerStyle={{
            bottom: "80px"
        }}
        carouselStyle={{
            borderColor: "black",
            borderWidth: "10px"
        }}
    />
</Playground>

## Props

Slider Component accepts props from all classes below.

### Main Props
#### Props Table
| Prop Name | Required Level | Type |
| --- | --- | --- |
| width | Needed |  number |
| height | Needed |  number |
| images | Needed |  string[] |
| maxSwipeThreshold | Optional |  number |
| minSwipeThreshold | Optional |  number |
| swipeTimeThreshold | Optional |  number |

#### Props Explaination
| Prop Name | Explaination |
| --- | --- |
| width | Widht of the carousel |
| height | Height of the carousel |
| images | String array of image urls |
| maxSwipeThreshold | Maximun pixel swipe by user to trigger a swipe action |
| minSwipeThreshold | Used with `swipeTimeThreshold`, if the user swipe such `minSwipeThreshold` amount of pixel within `swipeTimeThreshold` milliseconds, a swipe action is triggered |
| swipeTimeThreshold | Milliseconds. Used with `swipeTimeThreshold`, if the user swipe such `minSwipeThreshold` amount of pixel within `swipeTimeThreshold` milliseconds, a swipe action is triggered |

### Buttons Props
#### Props Table
| Prop Name | Required Level | Type |
| --- | --- | --- |

#### Props Explaination
#### Props Table
